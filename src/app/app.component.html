ngx-tooltip-ease

This library offers a simple, performant, and responsive tooltip, while leaving the styling control to the user. This library supports custom and build-in animations, multi-lines tooltip, and a custom offset space.

Simple tooltip

Tooltips are responsive

Resize the window and the tooltips will adapt their position if there is not enough space to the left/right/bottom/top. No resize event is attached.

Style & line return

Long tooltip text with carriage return, where you want it. Feel free to override default style class 'ngx-tooltip' with your own creativity.
